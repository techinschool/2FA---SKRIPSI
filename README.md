## Prerequisite:
1. Node.js to be installed.
2. Ionic cli:
```
npm install -g @ionic/cli
```

## How to use:
1. clone the project.
2. cd into the project.
3. install the dependencies:
```
npm install
```
4. run the project:
```
ionic serve
```
5. That's it, enjoy.
